The following has been used:
```html
<!DOCTYPE html>

<html lang="">
  <head>
    <title>
      <!--- text here -->
    </title>
    <link rel="stylesheet" href="{your file name}.css"> <!-- css = cascading style sheet -->
  </head>
  <body>
    <h1>
      <!-- text here -->
    </h1>
    <p><!-- more text here -->
    <br> <!-- line breaks -->
    <br>
    <!-- even more text here --></p>
    <ul>
      <li>
        <a href="<!-- the link to be redirected to -->" target="_blank <!-- ensures the link gets opened in a spcified way -->" title="<!-- text that will appear after hovered over the link (after about ~2.5 seconds) -->" id="<!-- the unique identifier to use in CSS and script files -->">
          <!-- what the text embedded with the link should appear as -->
        </a>
      </li>
    </ul>
  </body>
</html>
```
